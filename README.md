# image-to-password

🇷🇴 Generarea unei parole folosind o imagine
🇺🇸 Generating a password using an image


🇷🇴 Cum se foloseste:
Trage&plaseaza imaginea in fereastra cmd in timp ce scriptul ruleaza. Scriptul va citi imaginea si va genera o parola formata din A-Z, a-z sau A-Z, a-z, 0-9, !-/.

Informatii rapide:
-Latimea imaginii determina lungimea parolei;
-Inaltimea imaginii trebuie sa fie >= 2; ultima linie este folosita pentru a calcula un offset al valorilor utilizate la generarea caracterelor, iar penultima linie este folosita pentru a calcula ponderile;
-Se recomanda folosirea imaginilor mici formate din zgomot.

🇺🇸 How to use:
Drag&drop the image into the cmd window while the script is running. The script will read the image and return a password made out of A–Z, a–z or A–Z, a–z, 0–9, !–/.

Quick info:
-Image width determines password length;
-Image height must be >= 2; last row used to offset the values used to generate characters, and the second to last row used to compute weights.
-Use small images made of noise patterns
